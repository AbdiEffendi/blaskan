Overview

Great device support

Blaskan is a responsive theme built for smartphones, tablets as well as laptop and desktop computers.

Highly accessible

The web should be available to everyone and WAI-ARIA is supported as much as possible.

Multilingual

Translated into the following languages:

  • English
  • French
  • Dutch
  • Swedish

In detail

Supported browsers

  • Firefox
  • Safari
  • Chrome
  • Opera
  • Internet Explorer 7 and above
  • Android
  • Mobile Safari (iPhone/iPad/iPod touch)
  • Opera Mini

Included JavaScript libraries

  • Modernizr
  • Respond.js
  • Selectivizr
  • FitVids.js
  • Mobile boilerplate helper functions

Typography

Helvetica Neue, sans-serif

General body copy.

League Gothic

Site and page titles. Loaded with @font-face. Licensed under OFL.

(Optionally use Helvetica Neue, sans-serif via theme options.)
